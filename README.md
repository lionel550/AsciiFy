# AsciiFy
A little program to convert image a ascii art.

## Supported Formats
Supports all formats provided by the `stb_image` backend: JPEG, PNG, BMP, GIF, TGA, PSD, HDR, PIC, PNM

## Build & Run

### 1. Compile
Build the project using the provided Makefile:

```sh
make

```

### 2. Usage
Run the binary and provide the path to your image:

```sh
./out/asciify <path_to_image>

```
